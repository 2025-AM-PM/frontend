# 프로젝트 시작

다음 명령어 순서대로 실행하여 개발 서버를 시작합니다:

```bash
npm install
npm start
```

그 후 아래 주소로 접속하면 됩니다.
[http://localhost:3000/](http://localhost:3000/)
